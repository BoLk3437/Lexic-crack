## 💡 Почему этот проект бесплатный? (Манифест)💡

Мы глубоко убеждены, что **опенсорс должен оставаться открытым**. 

Сегодня платить за программы, которые на 100% написаны искусственным интеллектом, просто нет никакого смысла. Код должен принадлежать сообществу. Поэтому этот проект полностью бесплатен, открыт для любых изменений, форков и улучшений. 

> Пользуйтесь на здоровье, делитесь с друзьями и делайте мир опенсорса лучше! 🚀

## ⚙️ Как изменять сервера и профили?

Вся конфигурация максимально простая и происходит через файлы настроек. 

### Шаг 1. Выберите или создайте профиль
Все доступные проекты и серверы хранятся в папке **`profiles`** в корне программы. Вы можете добавлять туда свои файлы конфигураций.

### Шаг 2. Активируйте нужный профиль
Откройте файл `settings.json` в любом текстовом редакторе и измените значение ключа `"active_profile"` на название нужного проекта (без расширения файла).

**Пример настройки в `settings.json`:**
```json
{
  "active_profile": "УК + ПК + ДК"
}
